# NYCRPVIP Website This is the New York City Roleplay VIP website, ready for GitHub Pages. ## Files - `index.html` - Home page - `store.html` - Monthly VIP subscription store - `rules.html` - Server rules - `support.html` - Support page - `leaderboard.html` - Leaderboard page - `members.html` - Members page - `css/style.css` - Website styles - `js/script.js` - Discord invite, Stripe payment links, and app scripts - `images/` - Logo and background image - `manifest.webmanifest` - Installable app manifest - `service-worker.js` - Offline/app cache - `.nojekyll` - Tells GitHub Pages to serve the static files directly ## Publish On GitHub Pages 1. Create a new GitHub repository. 2. Upload all files from this folder into the repository. 3. Go to repository `Settings`. 4. Open `Pages`. 5. Under `Build and deployment`, choose: - Source: `Deploy from a branch` - Branch: `main` - Folder: `/root` 6. Click `Save`. GitHub will give you a public website link like: `https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPO-NAME/` ## Stripe Setup Open `js/script.js` and add your monthly Stripe Payment Links: ```js const STRIPE_MONTHLY_PAYMENT_LINKS = { "Bronze VIP": "", "Silver VIP": "", "Gold VIP": "", "Platinum VIP": "" }; ``` After the links are added, the Subscribe buttons will open Stripe checkout. ## Discord The Discord invite is already set to: `https://discord.gg/eBw23a4W`
